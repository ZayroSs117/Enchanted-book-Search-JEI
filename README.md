# Enchanted Book Search 1.0.6

Mod client Forge pour Minecraft 1.20.1 et JEI 15.20.0.130+.

- ajoute chaque livre du niveau minimum au véritable niveau maximum disponible ;
- récupère les limites configurées par Apotheosis avec le même hook que le mod utilise en jeu ;
- conserve l’ordre naturel du registre des enchantements ;
- réorganise chaque enchantement du niveau le plus faible au niveau le plus élevé ;
- corrige les suites du type `VIII, I, II, III...` provoquées par le livre maximum déjà ajouté par JEI ;
- restaure ensuite les livres spéciaux ajoutés par d’autres mods ;
- ajoute le nom traduit, l’identifiant et le chemin comme alias de recherche JEI ;
- ne modifie ni les enchantements ni le gameplay.

Place uniquement `enchanted-book-search-1.0.6.jar` dans le dossier `mods` du client.
